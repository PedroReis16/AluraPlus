# Alura Plus

## Sobre o projeto 
Alura Plus é um projeto desenvolvido em HTML e CSS, baseado em uma aula da plataforma Alura Cursos, onde foi apresentados os conceitos básicos sobre a estilização de página web em HTML e CSS.
